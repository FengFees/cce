# 创建集群<a name="cce_02_0236"></a>

## 功能描述<a name="section1686113493165"></a>

该API用于创建集群。

>![](public_sys-resources/icon-note.gif) **说明：**   
>-   调用该接口创建集群时，默认不安装icagent。若需安装icagent，请参照[帮助中心 \> 应用性能管理 \> 用户指南 \> Agent管理 \> 安装ICAgent](https://support.huaweicloud.com/usermanual-apm/apm_02_0013.html)。  
>-   此接口仅可以创建空集群（即只有控制节点master，没有用户节点node），可以在集群创建成功之后，[创建节点](创建节点.md)进行添加。  
>-   创建集群之前，要预先[创建vpc](创建VPC.md)和subnet。（若已经存在可重复使用）  
>-   一个账户默认只能创建 5 个集群，可以  [提交工单](https://console.huaweicloud.com/ticket/?#/ticketindex/createIndex)  增加配额。
