# 获取指定项目下的集群<a name="cce_02_0239"></a>

## 功能介绍

该API用于获取指定项目下所有集群的详细信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CCE&api=ListClusters)中调试该接口。

## URI

GET /api/v3/projects/\{project\_id\}/clusters
