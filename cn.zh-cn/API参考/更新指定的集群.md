# 更新指定的集群<a name="cce_02_0240"></a>

## 功能描述<a name="section1686113493165"></a>

该API用于更新指定的集群。

>![](public_sys-resources/icon-note.gif) **说明：**   
>-   当前仅支持更新spec下的description字段，即集群的描述。  

## URI<a name="section8403243161416"></a>
