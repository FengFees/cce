# 更新指定的集群<a name="cce_02_0240"></a>

## 功能介绍<a name="section1467313417555"></a>

该API用于更新指定的集群。

>![](public_sys-resources/icon-note.gif) **说明：** 
>集群管理的URL格式为：https://Endpoint/uri。其中uri为资源路径，也即API访问的路径。

## 调试<a name="section15674164111559"></a>

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CCE&api=UpdateCluster)中调试该接口。

## URI<a name="section96749419552"></a>
